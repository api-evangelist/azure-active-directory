# Azure Active Directory (azure-active-directory)
Microsoft Azure Active Directory (Azure AD) is Microsoft's cloud-based identity and access management service, which helps employees sign in and access resources.

**URL:** [Visit APIs.json URL](https://azure.microsoft.com/en-us/services/active-directory/)
